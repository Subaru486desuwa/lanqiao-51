# 蓝桥杯单片机（IAP15F2K61S2）练习工程

基于 **STC IAP15F2K61S2** 的蓝桥杯嵌入式赛道（51单片机方向）练习代码，使用 Keil C51 开发。

## 目录结构

### 单模块练习

| 目录 | 说明 |
|------|------|
| `LED灯/` | LED 流水灯、独立控制 |
| `BUZZ与delay/` | 蜂鸣器与延时函数 |
| `DS1302/` | DS1302 实时时钟 |
| `DS18B20/` | DS18B20 温度传感器（单总线） |
| `EEPROM/` | AT24C02 EEPROM 读写（I2C） |
| `AD_DA/` | PCF8591 ADC/DAC 模数转换（I2C） |
| `串口通讯/` | UART 串口通信 |
| `超声波/` | 超声波测距（HC-SR04） |
| `数码管显示圆周率前七位/` | 数码管动态扫描 |
| `定时器1定时1ms频率显示（修改3增加子界面swit回显+DAC)/` | 定时器综合应用（频率测量 + 子界面 + DAC） |
| `蓝桥杯全模块测试例程/` | 所有外设模块完整测试（DS1302/DS18B20/EEPROM/AD_DA/串口/超声波） |
| `大模板-自写/` | 自写综合模板（按键+数码管+LED+各外设） |
